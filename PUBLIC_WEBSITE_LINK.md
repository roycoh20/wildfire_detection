# Public Website Link

After publishing this folder with GitHub Pages, open the site from any computer at:

```text
https://roycoh20.github.io/wildfire_detection/
```

The local website entry point is:

```text
docs/index.html
```

## To Make The Link Live

1. Upload or push this project to:

```text
https://github.com/roycoh20/wildfire_detection
```

2. In GitHub, open the repository settings.
3. Go to **Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select branch **main**.
6. Select folder **/docs**.
7. Click **Save**.

GitHub usually publishes the site within a few minutes.
