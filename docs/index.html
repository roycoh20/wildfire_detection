Real-Time Edge Wildfire Detection System

Jetson-based wildfire detection using vision-language AI, semantic classification, and edge storage.

A real-time edge-AI project by Roy Cohen and Itay Hovav, developed at the Technion. The system analyzes live camera input on an NVIDIA Jetson Orin Nano, generates scene descriptions using a vision-language model, classifies them as wildfire or no-wildfire, and saves evidence for human verification.
